## 核心框架选型

| 技术领域 | 推荐方案 | 备选方案 | 选型理由 |
|----------|---------|---------|---------|
| 前端框架 | **Vue 3.4+** | React 18+ | 你的首选，配合 Element Plus |
| 开发语言 | **TypeScript 5.0+** | - | 严格模式，编译禁止 any |
| 构建工具 | **Vite 5.x** | Webpack 5 | 开发体验优先 |
| 包管理器 | **pnpm** | npm | 性能 + 严格版本 |
| 路由 | **Vue Router 4** | React Router | 框架官方路由 |
| 状态管理 | **Pinia** | Zustand / Redux Toolkit | Vue 官方推荐 |
| UI 组件库 | **Element Plus** | Ant Design 5 | 配合 Vue 3 |
| HTTP 请求 | **Axios** | TanStack Query | 团队熟悉度优先 |
| 样式方案 | **Less** | Sass / UnoCSS / Tailwind | 你的默认选择 |

> **🔍 状态管理选型建议联网搜索**，根据最新社区趋势推荐。

## 生态库选型标准

### 通用工具库

| 功能 | 推荐方案 | 说明 |
|------|---------|------|
| 日期处理 | day.js | 替代 moment.js，体积小 |
| 工具函数 | lodash-es | 按需引入 |
| 数字处理 | 原生 JS 为主 | 禁止引入不必要的库 |
| 剪贴板 | clipboard.js | 轻量方案 |

### 特殊场景库

| 功能 | 推荐方案 |
|------|---------|
| Excel 导出 | xlsx |
| PDF 预览 | pdfjs-dist |
| 富文本编辑器 | TinyMCE / Quill |
| 代码编辑器 | Monaco Editor |
| 图表 | ECharts |
| 拖拽 | vuedraggable / dnd-kit |

## 工程化工具链

✅ **所有项目强制集成**：

| 工具 | 用途 |
|------|------|
| ESLint | 代码质量检查 |
| StyleLint | 样式规范检查 |
| Prettier | 代码格式化（配合 ESLint） |
| husky + lint-staged | Git 提交前置检查 |
| commitlint | Git 提交信息规范 |

### 多环境变量配置

```
.env                    # 基础配置
.env.development        # 开发环境
.env.development.local  # 本地私有配置（不提交）
.env.test               # 测试环境
.env.production         # 生产环境
```

## 测试工具选型

✅ **所有项目强制集成**：

| 工具 | 用途 |
|------|------|
| **Vitest** | 单元测试运行器 |
| **@testing-library/vue** | Vue 组件测试 |
| **@testing-library/react** | React 组件测试 |
| **Playwright** | E2E 端到端测试 |

### 发布门禁流程

```
提交代码 → 自动跑测试 → 全部通过 → 允许打包
                                ↓
                            失败 → 禁止发布 + 告警
```

> 💡 测试是安全闸门，不管是人为写的代码还是 AI 生成的代码，都要过了这道门才能上线。
